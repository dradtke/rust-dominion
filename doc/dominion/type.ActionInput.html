pub enum ActionInput {
    Discard(Card),
    Trash(Card),
    Gain(Card),

    Repeat(Card, fn(uint) -> Vec<ActionInput>),

Input parameters for card plays.

Variants

Discard

Discard a card.

Trash

Trash a card.

Gain

Gain a card.

Confirm

Confirm an effect, i.e. discarding your deck with Chancellor.

Repeat

Repeat an effect, i.e. with Throne Room.

The first parameter is the card to repeat, and the second is a function from play iteration (starting with 0 and increasing by one each time the card is repeated) to the input for that card.
